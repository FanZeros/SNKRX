# 蛇蛇小队 - 角色与职业组合手册

## 职业（组合）一览

职业是角色的标签，集齐一定数量的同职业角色可激活组合效果。

| 职业 | 激活条件 | 效果 |
|------|---------|------|
| 战士 | 3/6 | 友方战士防御 +25/+50 |
| 游侠 | 3/6 | 友方游侠攻击时有 8%/16% 几率释放弹幕 |
| 法师 | 3/6 | 敌人防御 -15/-30 |
| 爆破手 | 3/6 | 友方爆破手范围伤害和范围 +15%/+25% |
| 盗贼 | 3/6 | 友方盗贼有 15%/30% 暴击几率，造成 4 倍伤害 |
| 治愈师 | 2/4 | 生成治疗球时有 +15%/+30% 几率额外生成 +1 个治疗球 |
| 附魔师 | 2/4 | 所有友军伤害 +15%/+25% |
| 构造师 | 2/4 | 构造物伤害和持续时间 +25%/+50% |
| 灵能师 | 2/4 | 灵能球总数 +2/+4，每个灵能师额外 +1 个球 |
| 诅咒师 | 2/4 | 友方诅咒师最大诅咒目标 +1/+3 |
| 力场师 | 2/4 | 所有友军击退力 +25%/+50% |
| 虫群师 | 2/4 | 仆从生命 +1/+3 |
| 虚空使 | 2/4 | 友方虚空使持续伤害 +20%/+40% |
| 术士 | 2/4/6 | 术士每 4/3/2 次攻击重复一次攻击 |
| 佣兵 | 2/4 | 敌人死亡时有 +8%/+16% 几率掉落金币 |
| 探索者 | 1 | 每个激活的职业给予友方探索者 +15% 攻速和伤害 |

## 组队建议

### 按职业凑组合

**容易凑的 3 人组合（3/6 激活）：**
- **战士**：剑士(1)、法外者(2)/野蛮人(2)/侍从(2) → 高地战士(4)/剑刃(4)/主宰(3)
- **法师**：魔术师(1)、法师(2)/冰霜法师(2)/时空法师(2) → 元素师(3)/魔剑士(3)/烈焰法师(3)
- **游侠**：弓箭手(1)、双枪手(2)/哨兵(2) → 弹幕射手(3)/炮手(4)/腐化者(4)
- **盗贼**：侦察兵(1)、法外者(2)/双枪手(2)/小丑(2)/驯兽师(2) → 刺客(3)/魔剑士(3)
- **爆破手**：法师(2)/轰炸手(2) → 元素师(3)/烈焰法师(3) → 剑刃(4)/炮手(4)/瘟疫医生(4)

**容易凑的 2 人组合（2/4 激活）：**
- **附魔师**：风暴编织者(3) + 侍从(2)/时空法师(2)/精灵(4)/苦修者(3)
- **治愈师**：牧师(1) + 雕刻师(2)/灵能守护者(3)/精灵(4)/祭司(4)
- **虚空使**：冰霜法师(2) + 刺客(3)/祸根(3)/瘟疫医生(4)/女巫(2)
- **虫群师**：驯兽师(2) + 宿主(3)/腐化者(4)/感染者(3)
- **力场师**：主宰(3) + 贤者(2)/弹幕射手(3)/念动师(4)/守望者(4)/弹射者(2)
- **术士**：奥术师(1) + 灵能者(2)/女巫(2)/沉默者(2)/赌徒(3) → 工匠(3)/火山术士(4)/守望者(4)
- **佣兵**：商人(1) + 矿工(2)/赌徒(3)/高利贷者(3)/盗贼(4)
